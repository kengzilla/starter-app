# Shared components (`me-*`)

| Component     | Selector     | Key inputs                          | Status   |
| ------------- | ------------ | ----------------------------------- | -------- |
| Button        | `<me-button>` | `variant`, `loading`, `disabled`, `type` | Shipped  |
| Card          | `<me-card>`   | `interactive`, `active`            | Planned  |
| Chip          | `<me-chip>`   | `variant`                          | Planned  |
| Input shell   | `<me-input-shell>` | `label`, `helperText`, `required`, `invalid` | Planned |

Request a design review before adding new primitives to this catalog.
